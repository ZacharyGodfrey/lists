# Lists

A site for creating lists

## Running locally

- Run `npm i` to install dependencies
- Run `npm run local` to build the site and run it on localhost

## Hosting

Hosted on Netlify at [https://zg-lists.netlify.com](https://zg-lists.netlify.com)
